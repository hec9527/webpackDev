# webpack搭建脚手架

脚手架包括react、typescript、babel、antd、以及各种lint

## use

```shell
npm i
npm start
```

## loader

- `ts-loader`
- `style-loader`、 `css-loader`、 `postcss-loader`
- `file-loader`、 `url-loader`
- `sass-loader`
- `less-loader`
